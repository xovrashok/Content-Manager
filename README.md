# Content Manager — Тестове завдання

Цей репозиторій містить реалізацію тестового завдання для позиції **Content Manager** від Lotoplay.  

Макет лендінгу був наданий у файлі `Lotoplay-testove.png`.  

Сторінка максимально наближена до макета та включає:
- Функціональне меню та кнопки.
- Дві форми з базовою перевіркою введених даних: 
  - Контактна форма для зв’язку.
  - Модалка для замовлення квитків.
- Адаптивний дизайн для мобільних пристроїв.

## **Функціонал**

1. **Навігація**  
   Кнопки меню переходять до відповідних секцій сторінки.

2. **Контактна форма**  
   - Перевірка заповнення полів: ім’я, email та повідомлення.
   - Вивід повідомлення про помилку або успішну відправку прямо на сторінці.

3. **Модалка “Замовити квиток”**  
   - Відкривається по кліку на будь-яку кнопку "Замовити квиток".
   - Перевірка полів: ім’я, email, кількість квитків.
   - Можливість закриття модалки через кнопку “×” або клік на overlay.

4. **Адаптивність**  
   - Структура сторінки підлаштовується під ширину екрана.
   - Мобільна версія: меню стає вертикальним, контент та таблиці підлаштовуються під маленькі екрани.

---

## **Запуск проекту**

1. Клонуйте репозиторій або скачайте ZIP-архів.
2. Відкрийте `index.html` у браузері.
3. Для перевірки роботи форм — заповніть поля та натисніть “Відправити” або “Замовити квиток”.
